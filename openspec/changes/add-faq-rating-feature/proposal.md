# 变更：为 FAQ 系统添加用户评分功能

## 原因
当前的 FAQ 系统缺乏用户反馈机制，无法了解哪些文章对用户有帮助。添加用户评分功能可以让用户对 FAQ 文章进行点赞或点踩，帮助识别最有用的内容，并为内容优化提供数据支持。

## 变更内容
- 为 FAQ 文章添加点赞/点踩评分功能
- 在文章详情页显示评分统计信息
- 在文章列表页显示评分摘要
- 添加用户评分历史记录防止重复评分
- 为管理员提供评分数据分析界面

## 影响
- 受影响的规范：faq-system
- 受影响的代码：
  - 数据库模式需要添加评分相关表
  - FAQ API 需要添加评分相关端点
  - 前端组件需要添加评分 UI 元素
  - 管理界面需要添加评分数据分析功能
- 破坏性变更：无
